Goyal Borewells Website - Upload Ready

Admin Panel:
Open yourdomain.com/#admin

Admin Features:
- Product photo upload / image URL
- Product name, description, rate, unit, MOQ, SKU, material, category
- Category image upload / image URL
- Catalogue, gallery, enquiries, content, theme

Supabase:
1. Run supabase/schema.sql in Supabase SQL editor.
2. Create admin user in Supabase Authentication.
3. Add Supabase URL and anon key in assets/js/config.js.
4. Deploy folder on Vercel.
